Installation

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install decent

Manual installation:

::

    $ git clone https://github.com/xeroc/python-decent/
    $ cd python-decent
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
